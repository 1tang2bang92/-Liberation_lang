# bok_lang / 광복어(?)
***
만든날이 광복절이라 bok lang(복어)로 만듦.

### feature / 특징
* ; 없음
* 문단 {} 로 구분
* 타입 추론(에정)
* smart pointer (예정)
* 버퍼 안정성(예정)
* utf-8 코딩(예정)
*

* 절차지향 -> 함수, 객체 예정 (예정)
